版本：v1.1  
日期：2026-05-18  
定位：高考后专业选择场景的轻量职业兴趣探索产品  
输出格式：Markdown

## 3. 产品目标与用户路径

### 3.1 目标用户

- 高三毕业生；
- 高一、高二提前做专业探索的学生；
- 陪同选专业的家长；
- 学院/招生咨询/生涯教育活动中的潜在用户。

### 3.2 产品目标

| 目标 | 解释 |
|---|---|
| 吸引 | 用“未来专业人格卡”形成类似 MBTI 的传播入口 |
| 解释 | 帮学生把兴趣翻译成专业大类 |
| 转化 | 引导学生查看深度专业报告、预约咨询、参加专业解读活动 |
| 降风险 | 不承诺“最适合”，只给“优先探索” |

### 3.3 页面流程

```text
首页
→ 题前说明
→ 18道大学场景题
→ 生成动画
→ 未来专业人格卡
→ 专业大类推荐
→ 深度报告/咨询入口
```

### 3.4 关键指标

| 指标 | 建议目标 |
|---|---|
| 首页开始率 | ≥ 35% |
| 测试完成率 | ≥ 75% |
| 结果页停留 | ≥ 45 秒 |
| 分享率 | ≥ 8% |
| 深度报告点击率 | ≥ 15% |
| 留资/预约率 | 视渠道另设 |


## 4. 入口文案与页面文案

### 4.1 首页主推版本

**主标题**  
> 高考后，你适合先看哪些专业大类？

**副标题**  
> 18道大学场景题，生成你的「未来专业人格卡」：看看哪些专业方向更值得你优先探索。

**按钮**  
> 开始生成我的专业人格卡

**底部小字**  
> 本测试用于专业兴趣探索，不代表能力评价，也不构成志愿填报建议。

### 4.2 更偏传播的版本

**标题**  
> 你是哪种未来专业人格？

**副标题**  
> 像热门人格测试一样好玩，但结果更贴近高考后的大学专业选择。

### 4.3 题前说明

> 接下来你会看到 18 个大学专业场景。  
> 每题请选择你最想负责的一件事，以及第二想负责的一件事。  
> 不需要考虑你现在会不会，只看你愿不愿意尝试。

### 4.4 结果页免责声明

> 这不是心理诊断，也不是志愿填报结论。它只是在帮你发现：哪些大学学习任务、专业大类和职业出口，更可能让你愿意投入。


## 5. 产品命名

推荐名称：

> **未来专业人格卡**

备选名称：

| 名称 | 评价 |
|---|---|
| 未来专业人格卡 | 最推荐，兼顾传播和高考专业场景 |
| 专业兴趣人格卡 | 更严肃，但传播感略弱 |
| 高考专业探索卡 | 稳妥，但娱乐性偏低 |
| 大学专业人格测试 | 好懂，但“测试”二字略严肃 |
| 未来职业人格卡 | 适合职业场景，但本版不如“专业人格卡”贴切 |

不建议：

- MBTI职业版
- 高考MBTI
- 测出你最适合的专业
- 你的命定职业

## 7. 首版“热门专业大类50”选择口径

本文件不是官方热门排名。这里的“热门”是产品首版口径，综合四类信号：

1. **高考可见度/招生覆盖**：优先参考附件里的“子类数”，它可以作为专业大类在招生和专业名称中可见度的近似指标；
2. **产业与就业趋势**：优先纳入 AI、软件信息、智能制造、新能源、医疗健康、金融、法律、教育、传媒、电商、供应链等方向；
3. **学生理解成本**：优先选学生和家长能快速理解、能和未来职业联系起来的大类；
4. **RIASEC覆盖平衡**：避免只推荐计算机、金融、医学，也要覆盖表达、助人、组织、工程、数据等不同兴趣类型。

特别说明：  
**工科试验班类、理科试验班类、人文社科试验班类、经济管理试验班类、综合试验班类**在高考填报中很重要，但它们更像“招生包/培养包”，不是稳定的兴趣方向。因此本版不把它们放入“职业人格主推荐50”，而是在产品中作为“院校填报解释模块”单独处理：当学生看到某校试验班时，系统应展开它包含的具体专业大类，再进入匹配逻辑。

## 8. 算法与映射关系

### 8.1 两层计分：先判人格，再推专业

第一层是 **RIASEC 兴趣人格分**，用于生成分享卡上的人格名称。

```text
每题选择：
- 最想做：该选项对应维度 +2
- 第二想做：该选项对应维度 +1

18题总分固定为 54 分。
R/I/A/S/E/C 六维分别累计。
取最高两个维度作为主结果，如 I-C = 数据侦探。
```

第二层是 **专业大类提示分**，用于让推荐更贴近高考报专业。

每个选项除了一个 RIASEC 维度，还配置若干个专业大类标签。用户选择该选项时，对这些专业大类加提示分。

```text
最想做选项：该选项绑定的专业大类各 +1.0
第二想做选项：该选项绑定的专业大类各 +0.5
```

最后每个专业大类的推荐分建议这样算：

```text
专业推荐分 =
0.65 × RIASEC匹配分
+ 0.35 × 题目选项专业提示分
```

其中 RIASEC 匹配分可以先用简单规则：

```text
学生第1维度命中专业标签：+5
学生第2维度命中专业标签：+3
学生第3维度命中专业标签：+1
```

这样做的好处是：  
**人格结果保持稳定、易分享；专业推荐又能根据学生在题目中反复选择的具体场景做细化。**

### 8.2 置信度规则

| 分数情况 | 展示策略 |
|---|---|
| 第一名比第二名高 5 分以上 | 主线明显，结果页写“你的主色很清晰” |
| 第一名和第二名差 0–4 分 | 双核心人格，不强行说单一类型 |
| 第二名和第三名差 0–2 分 | 展示“潜在第三色” |
| 最高分与最低分差小于 8 分 | 进入“未来多面手”，给3条探索线 |
| 某维度很低 | 不写“不适合”，只写“当前吸引力较低” |

### 8.3 结果页推荐数量

第一屏不要给太多：

```text
优先探索专业大类：3个
可以继续了解：3个
热门职业出口：3-5个
两周体验任务：1个
```

深度报告再展开到 8-12 个专业大类。

## 10. 结果页模板

### 示例：I-C「数据侦探」

**你的未来专业人格：数据侦探 I-C**

你容易被“数字背后的答案”吸引。相比只听别人下结论，你更愿意自己查证据、建模型、看规律。你适合优先探索需要数据、逻辑、系统和判断力的大学专业。

- **优先探索专业大类**：计算机类、统计学类、数学类、金融学类、管理科学与工程类、电子信息类
- **热门职业出口**：数据分析师、算法工程师、风控建模、商业分析师、量化研究、AI产品经理
- **不要误解**：这不代表你只能学计算机；如果你喜欢金融、医学、教育或城市议题，也可以选择“数据 + 行业”的复合路径。
- **两周体验任务**：用公开数据做一张图表，写出3个发现和1个建议。

### 示例：A-E「内容造风者」

**你的未来专业人格：内容造风者 A-E**

你对表达、创意和影响力更敏感。你不只是喜欢创作，也会在意内容能不能被看见、被传播、被转化成真实影响。

- **优先探索专业大类**：新闻传播学类、设计学类、戏剧与影视学类、电子商务类、旅游管理类、工商管理类
- **热门职业出口**：新媒体运营、品牌策划、广告创意、文旅策划、内容创业、用户增长
- **不要误解**：这不是“只适合艺术”。你也可以探索商业、平台、旅游、教育里的内容与品牌岗位。
- **两周体验任务**：为一个大学专业或校园活动设计一张海报、3条标题和一个30秒短视频脚本。

## 13. 合规与风险控制

### 13.1 MBTI 相关

产品名、页面标题、海报、按钮、URL 中不要使用：

- MBTI职业版
- 高考MBTI
- MBTI专业测试
- 16型职业人格MBTI
- Myers-Briggs 相关标识、图标、题目或报告模板

可以在 FAQ 中解释：

> 本测试不是 MBTI，也不使用 MBTI 量表。我们只是借鉴“人格卡”这种好理解、好分享的呈现方式；底层使用职业兴趣模型来做专业探索。

### 13.2 RIASEC / O*NET 相关

可以使用 RIASEC 作为理论框架，但不要直接复制 O*NET 的题目、报告文本或职业库内容。当前文档中的 18 道题为原创场景题，建议上线前再做内部审题与学生认知访谈。

### 13.3 未成年人和个人信息

建议第一版采用：

- 不登录即可测试；


### 13.4 结果表达底线

不要写：

- “你最适合学这个专业”
- “你不适合某专业”
- “测出你的命定职业”
- “高考志愿填报结论”

建议写：

- “这些专业大类更值得你优先探索”
- “这反映的是当前兴趣倾向，不代表能力评价”
- “最终选择还要结合成绩、选科、院校、地区、家庭条件和真实体验”


## 14. 上线前验证计划

### 14.1 快速试测

建议先做 100–200 人小样本，重点看：

| 项目 | 判断标准 |
|---|---|
| 完成率 | 18题完成率是否 ≥ 75% |
| 题目理解 | 学生是否能理解每个选项 |
| 结果共鸣 | 学生是否觉得“像我” |
| 专业推荐 | 推荐专业是否太窄或太泛 |
| 分享意愿 | 结果卡是否愿意发给同学 |
| 家长接受 | 家长是否觉得不是玄学 |

### 14.2 重点检查

1. 是否过度推荐计算机、金融、医学；
2. 是否对女生/男生、文科/理科产生明显刻板印象；
3. 是否把“成绩好不好”混进兴趣判断；
4. 是否出现“学生想学A，但结果强推B”的明显反感；
5. 是否需要在题目前加“不需要考虑你现在会不会”。
